# oblivion-inspired-syntax theme

Just a theme inspired by the Gedit Oblivion syntax coloring.
It's not exactly equal as atom do not detect all syntax element like Gedit.

Currently done theming:
* HTML
* CSS
* Javascript
* LESS
* PHP
